# BizzPlugin Options Framework ডকুমেন্টেশন

BizzPlugin Options Framework ডকুমেন্টেশনে স্বাগতম! এই ডকুমেন্টেশন আপনাকে ফ্রেমওয়ার্কটি কার্যকরভাবে বুঝতে এবং ব্যবহার করতে সাহায্য করবে।

## সূচিপত্র

1. [ভূমিকা](introduction.md) - ফ্রেমওয়ার্ক কী এবং কেন ব্যবহার করবেন
2. [শুরু করার নির্দেশিকা](getting-started.md) - নতুন ডেভেলপারদের জন্য দ্রুত শুরুর গাইড
3. [ফিল্ড টাইপ](field-types.md) - সব উপলব্ধ ফিল্ড টাইপের সম্পূর্ণ রেফারেন্স
4. [সেকশন এবং সাবসেকশন](sections-subsections.md) - আপনার সেটিংস কীভাবে সংগঠিত করবেন
5. [প্যানেল কনফিগারেশন](panel-configuration.md) - প্যানেল ব্র্যান্ডিং, রিসোর্স এবং প্রস্তাবিত প্লাগইন কনফিগার করা
6. [চেইনেবল API](chainable-api.md) - প্যানেল তৈরির জন্য আধুনিক চেইনেবল API
7. [ফিল্টার এবং হুক](filters-hooks.md) - ফ্রেমওয়ার্ক এক্সটেন্ড করার জন্য উপলব্ধ ফিল্টার এবং হুক
8. [REST API](api.md) - বাহ্যিক ইন্টিগ্রেশনের জন্য REST API এন্ডপয়েন্ট
9. [ওয়েবহুক](webhooks.md) - ওয়েবহুক কনফিগারেশন এবং ব্যবহার
10. [এক্সপোর্ট/ইমপোর্ট](export-import.md) - সেটিংস এক্সপোর্ট এবং ইমপোর্ট করা
11. [প্রিমিয়াম ফিচার](premium-features.md) - প্রিমিয়াম/প্রো ফিল্ড লকিং কীভাবে বাস্তবায়ন করবেন
12. [উদাহরণ](examples.md) - সম্পূর্ণ কোড উদাহরণ

## দ্রুত পর্যালোচনা

BizzPlugin Options Framework হল একটি শক্তিশালী, আধুনিক WordPress অপশন ফ্রেমওয়ার্ক যা ডেভেলপারদের প্লাগইন এবং থিমের জন্য সুন্দর এবং কার্যকরী সেটিংস প্যানেল তৈরি করতে দেয়।

### প্রধান বৈশিষ্ট্য

- **AJAX সেভ** - পেজ রিলোড ছাড়াই সেটিংস সংরক্ষণ
- **REST API** - অপশন পড়া এবং আপডেট করার জন্য সম্পূর্ণ REST API সাপোর্ট
- **ওয়েবহুক** - সেটিংস পরিবর্তন হলে বাহ্যিক সার্ভিসে নোটিফাই করুন
- **চেইনেবল API** - প্যানেল কনফিগারেশনের জন্য ফ্লুয়েন্ট, চেইনেবল মেথড
- **একাধিক ফিল্ড টাইপ** - বিল্ট-ইন ২০+ ফিল্ড টাইপ
- **সেকশন এবং সাবসেকশন** - সেটিংস যৌক্তিকভাবে সংগঠিত করুন
- **শর্তসাপেক্ষ ফিল্ড** - অন্য ফিল্ডের মানের উপর ভিত্তি করে ফিল্ড দেখান/লুকান
- **প্রিমিয়াম ফিল্ড সাপোর্ট** - প্রিমিয়াম ভার্সনের জন্য ফিল্ড লক করুন
- **এক্সপোর্ট/ইমপোর্ট** - সহজে সেটিংস ব্যাকআপ এবং রিস্টোর করুন
- **এক্সটেনসিবল** - কাস্টমাইজেশনের জন্য হুক এবং ফিল্টার
