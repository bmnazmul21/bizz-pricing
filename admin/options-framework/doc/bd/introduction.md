# ভূমিকা

## BizzPlugin Options Framework কী?

BizzPlugin Options Framework হল একটি আধুনিক, ডেভেলপার-বান্ধব WordPress অপশন ফ্রেমওয়ার্ক যা WordPress প্লাগইন এবং থিমের জন্য সেটিংস প্যানেল তৈরি সহজ করার জন্য ডিজাইন করা হয়েছে। এটি ডেভেলপার এবং শেষ ব্যবহারকারী উভয়ের জন্য একটি পরিষ্কার, স্বজ্ঞাত ইন্টারফেস প্রদান করে।

## কেন এই ফ্রেমওয়ার্ক ব্যবহার করবেন?

### ১. ডেভেলপার-বান্ধব

- **ন্যূনতম কোড**: মাত্র কয়েক লাইন কোডে একটি সম্পূর্ণ সেটিংস প্যানেল তৈরি করুন
- **চেইনেবল API**: প্যানেল তৈরির জন্য আধুনিক, ফ্লুয়েন্ট ইন্টারফেস
- **সুন্দর ডকুমেন্টেশন**: বিস্তৃত ডকুমেন্টেশন এবং উদাহরণ
- **টাইপ-সেফ**: বিল্ট-ইন সঠিক ডেটা স্যানিটাইজেশন এবং ভ্যালিডেশন

### ২. ফিচার-সমৃদ্ধ

- **২০+ ফিল্ড টাইপ**: সাধারণ টেক্সট ফিল্ড থেকে জটিল ইমেজ সিলেক্টর পর্যন্ত
- **AJAX সেভ**: সেটিংস সংরক্ষণের সময় পেজ রিলোডের প্রয়োজন নেই
- **REST API**: বিল্ট-ইন বাহ্যিক ইন্টিগ্রেশন সাপোর্ট
- **ওয়েবহুক**: সেটিংস পরিবর্তন হলে বাহ্যিক সার্ভিসে নোটিফাই করুন
- **এক্সপোর্ট/ইমপোর্ট**: সহজে সেটিংস ব্যাকআপ এবং রিস্টোর করুন

### ৩. আধুনিক UI/UX

- **রেসপন্সিভ ডিজাইন**: সব স্ক্রিন সাইজে কাজ করে
- **স্বজ্ঞাত নেভিগেশন**: সেকশন এবং সাবসেকশন সহ ট্যাব-ভিত্তিক নেভিগেশন
- **ভিজ্যুয়াল ফিডব্যাক**: পরিষ্কার সেভ স্ট্যাটাস ইন্ডিকেটর এবং নোটিফিকেশন
- **অ্যাক্সেসিবিলিটি**: WordPress অ্যাক্সেসিবিলিটি স্ট্যান্ডার্ড অনুসারে তৈরি

### ৪. এক্সটেনসিবল

- **WordPress হুক**: ব্যাপক ফিল্টার এবং অ্যাকশন হুক
- **কাস্টম ফিল্ড টাইপ**: সহজে আপনার নিজের ফিল্ড টাইপ যোগ করুন
- **প্যানেল-নির্দিষ্ট ফিল্টার**: অন্যদের প্রভাবিত না করে নির্দিষ্ট প্যানেল টার্গেট করুন

## এটি কীভাবে কাজ করে

ফ্রেমওয়ার্কটি দুটি প্রধান কম্পোনেন্ট নিয়ে গঠিত:

1. **Framework ক্লাস (`BizzPlugin_Framework`)**: প্যানেল পরিচালনা করে এবং AJAX/API রিকোয়েস্ট হ্যান্ডেল করে
2. **Panel ক্লাস (`BizzPlugin_Panel`)**: সেকশন এবং ফিল্ড সহ পৃথক সেটিংস প্যানেল প্রতিনিধিত্ব করে

### মৌলিক প্রবাহ

```
প্লাগইন → Framework ইনস্ট্যান্স তৈরি → প্যানেল তৈরি → সেকশন এবং ফিল্ড যোগ → ব্যবহারকারী সেটিংস সংরক্ষণ → AJAX/Webhook/API
```

## প্রয়োজনীয়তা

- WordPress 5.0 বা তার বেশি
- PHP 7.4 বা তার বেশি

## ইন্টিগ্রেশন পদ্ধতি

আপনি দুটি উপায়ে আপনার প্লাগইনে ফ্রেমওয়ার্ক ইন্টিগ্রেট করতে পারেন:

### পদ্ধতি ১: আপনার প্লাগইনে কপি করুন

`options-framework` ফোল্ডারটি আপনার প্লাগইন ডিরেক্টরিতে কপি করুন এবং লোডার ইনক্লুড করুন:

```php
require_once plugin_dir_path(__FILE__) . 'options-framework/options-loader.php';
```

### পদ্ধতি ২: ডিপেন্ডেন্সি হিসেবে

যদি Composer বা অনুরূপ ডিপেন্ডেন্সি ম্যানেজার ব্যবহার করেন, আপনি ফ্রেমওয়ার্কটি ডিপেন্ডেন্সি হিসেবে অন্তর্ভুক্ত করতে পারেন।
