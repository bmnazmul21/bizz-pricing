## Quick Overview

BizzPlugin Options Framework is a powerful, modern WordPress options framework that allows developers to create beautiful and functional settings panels for plugins and themes.

### Key Features

- **AJAX Save** - Settings save without page reload
- **REST API** - Full REST API support for reading and updating options
- **Webhooks** - Notify external services when settings change
- **Chainable API** - Fluent, chainable methods for panel configuration
- **Multiple Field Types** - 20+ field types out of the box
- **Sections & Subsections** - Organize settings logically
- **Conditional Fields** - Show/hide fields based on other field values
- **Premium Field Support** - Lock fields for premium versions
- **Export/Import** - Backup and restore settings easily
- **Extensible** - Hooks and filters for customization
